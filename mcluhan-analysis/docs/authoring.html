Authoring Review Tool - Instructions

Navigation

Use ← → arrow keys or click the navigation arrows. Use the dropdown to jump directly to any spread.

Text Selection

Each spread has multiple text options (A, B, C). Click View to preview how it renders on the page. Click Select to mark it as your chosen option. Press 1 2 3 to quickly switch between options, and Enter to select the currently viewed option.

Quality Rating

Rate each text option's quality using the colored buttons: 3 = excellent (include in all versions), 2 = acceptable (include in alternate versions), 1 = reject (do not include). Click a rating again to remove it.

Image Selection

When images are available, thumbnails appear at the bottom of the render area. Click a thumbnail to preview it, then click Select Image to lock your choice. Use [ and ] keys to cycle through image options.

Fonts

Use the three font dropdowns (Display, Body, Caption) above the render area to change typefaces. Click Reset to return to defaults. Font choices are saved automatically and included in the export.

Notes

Type review notes in the text area below the option cards. Notes are saved automatically per spread.

Export

Click Export to download a JSON file containing all your selections, quality ratings, font choices, and notes.
